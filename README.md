# L.U.R.K.E.R.

A roguelike inspired by S.T.A.L.K.E.R. and Escape from Tarkov:

```
Movement/Attack   :  NumPad or Arrow Keys or VIM keys
Fire              :  f
Inventory         :  i
Get Item          :  g
Drop Item         :  d
Look              :  / or Numpad 5
Character Screen  :  c
Message Log       :  v
Interact (Stairs) :  e
```
 
Music by Ethan Hamman
